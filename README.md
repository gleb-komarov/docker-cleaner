# Docker cleaner

Script to quick clean up your system of Docker files.

## Usage
```bash
#clone repository
git clone https://github.com/gleb-komarov/docker-cleaner.git
```
```bash
#change directory
cd docker-cleaner
```
```bash
#change access permissions 
chmod u+x main.sh
```
```bash
#run script
./main.sh
```
